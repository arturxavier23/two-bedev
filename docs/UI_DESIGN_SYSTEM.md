# 🎨 Design System — Two-BeDev

## Identidade Visual

O Two-BeDev utiliza uma identidade visual inspirada no tema de abelhas, transmitindo produtividade, colaboração e progresso.

## Paleta de Cores

| Nome | Hex | Uso |
|------|-----|-----|
| **Primary (Amarelo)** | `#F5A623` | Botões principais, XP, destaques |
| **Secondary (Azul Escuro)** | `#1A1A2E` | Backgrounds, textos principais |
| **Background** | `#FFFFFF` | Fundo das telas |
| **Surface** | `#F5F5F5` | Cards, containers |
| **Success** | `#4CAF50` | Acertos, conquistas |
| **Error** | `#F44336` | Erros, alertas |
| **Warning** | `#FF9800` | Avisos |

## Tipografia

| Estilo | Tamanho | Peso | Uso |
|--------|---------|------|-----|
| H1 | 28px | Bold | Títulos principais |
| H2 | 22px | SemiBold | Subtítulos |
| H3 | 18px | SemiBold | Seções |
| Body | 16px | Regular | Texto corrido |
| Caption | 12px | Regular | Legendas, hints |

## Espaçamentos

```
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
```

## Componentes de Gamificação

### Barra de Progresso (XP)
- Altura: 12px
- Border radius: 6px
- Cor de fundo: `#E0E0E0`
- Cor de preenchimento: `#F5A623`

### Badges
- Tamanho: 64x64px
- Border radius: 50% (circular)
- Sombra sutil para destaque

### Cards de Lição
- Padding: 16px
- Border radius: 12px
- Sombra: `0 2px 8px rgba(0,0,0,0.1)`
